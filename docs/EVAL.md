# Evaluation Strategy

## Gates
- Citation integrity: text claims should carry source locators in grounded mode.
- Safety blocks: prohibited ayurveda/ritual requests rejected.
- Regression: kundli and vaastu endpoints keep stable schema.
- Video policy: generated scripts include spoken/on-screen disclaimers.

## Nightly workflow target
- Unit + integration + eval suite.
- Red-team prompt set execution.
- Results logged with trend lines; no auto-prod deployment.
